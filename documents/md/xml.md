# /files/xml

XMLファイルを収めるディレクトリです。

## 抽象 Abstraction

Extensible Markup Language (XML) 1.0 (Fifth Edition)
https://www.w3.org/TR/xml/

## Introduction

### ディレクトリの命名

名前空間でわける。
バージョンが違う場合、1の場合はv1となるようにする。
XHTML1.0または1.1の場合、
/files/xml/xhtml/v1/sample.xhtml
のようになる。
versionが、1.0の場合はXMLのversionは1.0、2.0の場合はXMLのversionを1.1、というようにすることがあるかもしれない。
消し忘れたディレクトリがあった場合、残すと混乱するだけだと思っているので、警告や予告無く削除することがある。
名前空間の無いXMLを_nonamespaceディレクトリに置く。
